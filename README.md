### Prerequisites

- Node.js (v16 or higher)
- npm (v8 or higher) or yarn

### Getting Started

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd onboarding-flow
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn
   ```

3. **Start the development server**
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. **Open in browser**
   The application will be available at `http://localhost:5173`
